
# Сапёр × Дота

Игра сапёр с элементами Dota 2 — скиллы героев, система рангов и прокачка.

## Как запустить

1. Установи Node.js — https://nodejs.org
2. Открой папку проекта в VS Code
3. В терминале:

npm install
node server/index.js

4. Открой браузер: http://localhost:3000

## Что есть в проекте

- Регистрация и вход
- 3 этапа сапёра (9×9, 20×20, 50×20)
- Скиллы: CM Frost, WR Power Shot, TA Shield
- Прокачка скиллов между этапами
- Таймер и счётчик мин
- Система рангов и MMR
- Светлая и тёмная тема
